# Web Stack Debugging 2

This project focuses on permissions and running services as non-root users for better security.

## Task 0: Run software as another user
A Bash script `0-iamsomeoneelse` that runs `whoami` as a specified user.

## Task 1: Run Nginx as Nginx
A Bash script `1-run_nginx_as_nginx` that configures Nginx to run as the `nginx` user and listen on port `8080`.
