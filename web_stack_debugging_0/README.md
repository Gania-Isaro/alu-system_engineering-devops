# Web Stack Debugging #0

This project is about debugging a web stack. The goal is to get Apache running on a Docker container (`holbertonschool/265-0`) so that it returns a page containing `Hello Holberton` when querying the root.

## Tasks

### 0. Give me a page!
- [0-give_me_a_page](./0-give_me_a_page)
- Description: A Bash script that fixes the Docker container so that Apache serves a page containing "Hello Holberton" on port 80.
